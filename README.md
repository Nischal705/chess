## Chess Engine 

A lightweight, bitboard-based chess engine built with C++ and Raylib.


### Features
*   **Engine:** Optimized bitboard representation for fast move generation.
*   **AI Opponent:** Implementation of the Minimax algorithm with Alpha-Beta pruning.
*   **Game Modes:** Supports both Single Player (vs Engine) and Two-Player (Local) modes.
*   **Complete Rules:** Includes Castling, En Passant, and Pawn Promotion.


### Controls

* **Left Click & Drag:** move pieces
  
*  **Promotion Menu:** select piece when a pawn reaches the last rank
  

### Build Instructions:

* **Prerequisites:** C++ compiler and raylib

* **Clone** this repository
  
*  **Compile** with the command "make" in the terminal in the project directory
  
* **Run** with the command ".\chess.exe" in the terminal
